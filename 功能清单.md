## 七、文件类型支持

### 7.1 文件分类

| 分类 | 说明 | 预览支持 |
|------|------|----------|
| Folder | 文件夹 | ✅ |
| Image | 图片 (JPG/PNG/GIF/BMP/SVG) | ✅ |
| Docs | 文档 (PDF/TXT/MD/JSON/XML/CSV) | ✅ (部分) |
| Video | 视频 (MP4/WebM/OGG/MOV) | ✅ |
| Audio | 音频 (MP3/WAV/FLAC) | ⏳ |
| Other | 其他文件 | ❌ |

### 7.2 允许上传的扩展名

```
图片：.jpg, .jpeg, .png, .gif, .bmp, .svg, .webp
文档：.pdf, .txt, .md, .json, .xml, .csv, .doc, .docx, .xls, .xlsx, .ppt, .pptx
视频：.mp4, .webm, .ogg, .mov, .avi, .mkv
音频：.mp3, .wav, .flac, .aac
压缩包：.zip, .rar, .7z
代码：.js, .ts, .vue, .py, .java, .cs, .go, .rs
其他：.exe, .dll, .dmg, .apk
```
